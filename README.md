# go_workspace

## go test repository

試験的なものを試すリポジトリー

## リポジトリーのクローンに関して

gopath の直下に以下のように作成する。  
`$GOPATH/src/github.com/clone対象のRepositoryのgithubユーザ/`

このリポジトリーは以下  
`$GOPATH/src/github.com/humi3/`

\$GOPATH については、mac だった以下になっているはず
`/Users/ユーザ/go`
